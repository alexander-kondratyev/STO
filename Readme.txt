Типы запросов

1. Регистрация нового аккаунта (POST)

2. Отправка ОТП для подтверждения номера телефона/Входа в аккаунт (GET)

3. Подтверждение номера телефона (POST)

4. Вход в аккаунт (POST)

5. Обновление профиля (PUT)

6. Обновление аватарки (PUT)

7. История обслуживания с фильтрами (Дата, тип, авто) (GET)

8. Добавление авто в гараж (POST)

9. Обновление авто в гараж (PUT)

10. Удаление авто из гаража (DELETE)

11. Удаление аккаунта
